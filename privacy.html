Política de Privacidad — Fiore Automation

Última actualización: julio de 2026

«Fiore Automation» es una aplicación interna desarrollada y utilizada exclusivamente por Sandwichería FIORE Confitería (Lomas de Zamora, Buenos Aires, Argentina) para automatizar la publicación de contenido propio (fotografías de sus productos) en la cuenta de Instagram del propio negocio, mediante la API oficial de Meta.

1. Qué datos utiliza la aplicación

La aplicación accede únicamente a la cuenta comercial de Instagram y a la página de Facebook pertenecientes a Sandwichería FIORE Confitería, con el único fin de crear publicaciones en dicha cuenta. Las imágenes publicadas son fotografías de productos tomadas por el propio negocio.

2. Datos de terceros

La aplicación no recopila, almacena, procesa ni comparte datos personales de terceros. No tiene usuarios externos, no ofrece inicio de sesión a terceros, no accede a perfiles, comentarios, mensajes ni ninguna otra información de otras personas.

3. Almacenamiento

Las fotografías del negocio se almacenan en una cuenta privada de Google Drive del propio negocio y, de forma temporal (minutos), en un repositorio de GitHub durante el proceso de publicación. Las credenciales de acceso se guardan cifradas como secretos de GitHub Actions.

4. Eliminación de datos

Dado que la aplicación no recopila datos de terceros, no existen datos de usuarios que eliminar. Si tenés alguna consulta o creés que la aplicación posee algún dato tuyo, escribinos y lo atenderemos a la brevedad.
